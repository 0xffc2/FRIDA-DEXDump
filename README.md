# FRIDA-DEXDump

Is an Android Unpacker for fast and simple.
Implemented with search dex-magic in memory.

## Feature
1. support fuzzy search no-magic dex. (eg: baidu protect)
2. auto fill magic into dex-header.

## Usage
1. launch app.
2. python main.py.
3. check ./<pkg_name>/*.dex.
